# Text-Fast
Um joguinho feito em Html,CSS e Javascript, o objetivo é digitar a palavra
que aparecer na tela o mais rapido possivel, ele tem 5 rounds, e no final mostra 
a sua melhor velocidade e em qual palavra vc foi melhor

## Tecnologias
-HTML5
-CSS3
-JAVASCRIPT

## Como executar
<ol>
    <li>Clone o repositorio</li>
    <li>Abra o arquivo "index.html" no seu navegador</li>
</ol>

## Como jogar
Digite a palavra que aparece a cima o mais rapido possivel durante 5 rounds
